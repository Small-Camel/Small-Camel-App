### 1.2 登陆

#### 1. 接口名

GET onLogin?

##### PathParameter

| 参数名 | 参数     |
| ------ | -------- |
| code   | res.code |

##### Payload

```javascript
wx.login({
  success: function(res) {
    if (res.code) {
      //发起网络请求
      wx.request({
        url: "http://localhost:8080/onLogin",
        data: {
          code: res.code
        }
      });
    } else {
      console.log("登录失败！" + res.errMsg);
    }
  }
});
```

#### 2. 返回值

```json
{
  "openid":(String)唯一标识符
}
```
