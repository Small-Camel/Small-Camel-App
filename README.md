# 这是一个 WePY 架构下的小程序

请使用如下命令编译

```js
  npm install wepy-cli -g
  npm install
  wepy build
```


同时设置开发环境

> 对应关闭ES6转ES5选项，关闭。 **重要：未关闭会运行报错。**
> 
> 对应关闭上传代码时样式自动补全选项，关闭。 **重要：某些情况下漏掉此项也会运行报错。**
> 
> 对应关闭代码压缩上传选项，关闭。**重要：开启后，会导致真机computed,props.sync 等等属性失效。**
> 
> 对应不检查安全域名选项，开启。
